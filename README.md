# student-information-system-pyqt
This is a desktop application developed using Python and PyQt5.
It allows admins and students to manage academic information.

Features:
- Admin and student login
- Student registration
- Add, update and delete student data
- Course and grade management
- GPA calculation
- SQLite database

Technologies:
- Python
- PyQt5
- SQLite

How to run:
- Install PyQt5
- Run uygulama_v1.py
